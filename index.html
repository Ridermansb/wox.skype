Wox.skype

Wox plugin for skype

Performs search of Skype contacts and start a conversation.

When no query is performed (just typing sky), option to open skype appears.

Only work on x86 platform, see issue #1, so install x86 version of Wox
if you use a chocolatey, choco install wox --x86 -y -f --acceptlicense
